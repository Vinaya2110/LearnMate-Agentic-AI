# LearnMate – Agentic AI for Personalized Course Pathways

## Problem Statement
Students often struggle to identify the right learning path that aligns with their interests and career goals.

## Solution
LearnMate is an Agentic AI system that analyzes student interests, skill levels, and career aspirations to generate personalized learning pathways.

## Technologies Used
- IBM Cloud
- IBM watsonx.ai
- IBM watsonx Orchestrate
- RAG (Retrieval-Augmented Generation)
- IBM Granite / Foundation Models

## Features
- Personalized learning pathways
- Skill-based recommendations
- Duration estimation
- Capstone project suggestions
- Progress tracking

## Domains Supported
- AI/ML
- Cybersecurity
- Cloud Computing
- Frontend Development

## Architecture
Student → LearnMate Agent → Knowledge Base → Personalized Learning Pathway
